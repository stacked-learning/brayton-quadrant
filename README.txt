BRAYTON QUADRANT V2.0
=====================

Contents:
- BraytonQuadrant.html - Standalone HTML application
- images/ - Folder containing 14 PNG images

How to Use:
1. Extract this ZIP file to any location on your computer
2. Open BraytonQuadrant.html in a web browser (Chrome, Firefox, Edge, Safari)
3. The application will work completely offline

Requirements:
- Modern web browser with JavaScript enabled
- Internet connection ONLY for initial load of CDN libraries (React, Tailwind, KaTeX)
- After initial load, the application works offline with local images

Features:
- Interactive 4-quadrant layout (Compression, Combustion, Expansion, Exhaust)
- Hover effects to preview each section
- Click to view detailed information
- Toggle between Cycle Schematic and TS/PV diagrams
- LaTeX-rendered formulas
- Visual indicators for thermodynamic properties
- Responsive design for mobile and desktop

Color Scheme:
- Compression: Blue (#1A6FEA)
- Combustion: Dark Red (#D22E2E)
- Expansion: Light Red (#F45A5A)
- Exhaust: Pink (#E587A8)
